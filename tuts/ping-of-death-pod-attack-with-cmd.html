How To Perform Ping of Death Attack Using CMD And Notepad

so what you require?

  • notepad
  • CMD


how it works :

Not all computers can handle data larger than a fixed size. So, when a ping of death packet is sent from a source computer to a target machine, the ping packet gets fragmented into smaller groups of packets. One fragment is of 8 octets size. When these packets reach the target computer, they arrive in fragments. So, the target computer reassembles the malformed packets which are received in chunks. But, the whole assembled packet causes buffer overflow at the target computer. This buffer flow often causes the system crash making the system more vulnerable to attack. Once the system becomes more vulnerable to attack, it allows more attacks like the injection of a trojan horse on the target machine.

So lets start :

open notepad
Copy the following text on the notepad 
:loop
ping |IP Address| -l 65500 -w 1 -n 1
goto :loop
In the above command, replace with an IP address.
Save the the file as dos.bat
now double click on it and you will see a command prompt running with a lot of pings.
This command might not work on some systems.
